# hyyg
A generator for Haoyayi

[![gitter][gitter-image]][gitter-url]
[![NPM version][npm-image]][npm-url]

# Features

- 执行命令时，判断是否在项目根目录

# Install

```sh
  $ [sudo] npm install hyyg -g
```

# Usage

## 生成 model
```sh
  $ hyyg patientAddInfo -m
```

## 生成 controller
```sh
  $ hyyg patientManagement -c
```
